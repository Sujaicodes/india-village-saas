Introduction

GeoVillage API provides programmatic access to India's complete village-level geographical data — sourced from Census 2011 and the Local Government Directory (LGD). It covers the full administrative hierarchy: State → District → Sub-District → Village.

The base URL for all API requests is:

https://village-geo-api.vercel.app/api

Authentication

All API requests require an API key passed in the X-API-Key request header. You can get your free API key by creating an account.

curl https://village-geo-api.vercel.app/api/states \
  -H "X-API-Key: your_api_key_here"

GET /api/states

Returns a list of all states and union territories in India.

GET /api/villages/search

Search for villages by name using fuzzy matching. Returns matching villages across all states.

ParameterTypeRequiredDescription
qstringrequiredSearch query (village name)
state_codestringoptionalFilter by state code
limitnumberoptionalMax results (default: 20)
curl "https://village-geo-api.vercel.app/api/villages/search?q=Pangi&state_code=02" \
  -H "X-API-Key: your_api_key"
